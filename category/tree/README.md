

### 一、二叉树
```


首先  二叉树 和 链表是一样的、 都是动态数据结构

二叉树 之所以叫二叉树、是因为、从根节点开始 每个节点只能有两个子树
同时 每个节点只能有一个父节点

有10个叉、就是 十叉树

二叉树具有天然 递归结构
每个节点的左子树 也是一个二叉树
每个节点的右子树 也是一个二叉树

二叉树不一定是满的

一个节点也是二叉树

```

### 二、二叉搜索树

```

二叉搜索树 是二叉树、有二叉树的所有特性


    二分搜索树的每个节点的值都要大于、它的左子树的所有节点的值
    二分搜索树的每个节点的值都要小于、它的右子树的所有节点的值
    存储的元素 必须有可比较性
    二分搜索树


二叉搜索树 不包含重复元素
    如果想 包含重复元素的话、 只需要定义:
        左子树小于等于节点、 或者


二叉搜索树 添加新元素

    1. 添加元素的非递归实现  和链表非常相似 
    2. 添加元素的递归实现


二分搜索树 是否包含某个元素

    1. 递归实现


遍历:
    遍历操作就是把所有节点都访问一遍

前中后序遍历都通常都使用递归、前序遍历也可以使用循环+栈的形式

前序遍历: 先遍历节点的值、然后左子树、最后遍历右子树
中序遍历: 先遍历节点的左子树、然后遍历节点的值、最后遍历右子树
后续遍历: 先遍历节点的右子树、然后遍历左子树、最后遍历节点的值
    
层序遍历: 从根开始、一层一层的遍历、 会遍历每一层的 左右子树后、
        才能遍历下一层的值、通常使用循环+队列(或链表)



```
